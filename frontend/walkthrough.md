# Walkthrough - College Social Media App

## Chat Feature
The chat feature allows users to communicate with each other in real-time. It includes a list of users, a message area, and an input field.

### Features
- **User List**: Displays all available users for chat.
- **Real-time Messaging**: Messages are sent and received instantly (simulated).
- **AI Chatbot**: A "My AI 🤖" bot that responds to user queries with a simulated typing indicator.
- **Typing Indicators**: Visual cues when the AI is processing a response.

### Verification
- **Chat Interface**: Verified that the chat interface loads correctly.
- **Sending Messages**: Verified that messages can be sent to other users and the AI bot.
- **AI Response**: Verified that the AI bot responds to messages with a typing indicator.

## Connect Page
The Connect page allows students to find and connect with peers based on major, year, and interests.

### Features
- **Search & Filter**: Users can search for students by name or filter by major and year.
- **Student Cards**: Displays student profiles with "Connect" buttons.

### Verification
- **Page Load**: Verified that the Connect page loads and displays student cards.

## Study Groups Page
The Study Groups page lists available study groups that students can join.

### Features
- **Group Listings**: Displays groups with subject, members count, and meeting times.
- **Join Functionality**: Users can join groups (simulated).
